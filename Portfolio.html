Project 1 - Song Genre Predictions

September, 2024 - December, 2024

Project Repository

In this project me and my team used various utilized various machine learning methods such SVMs (Support Vector Machines), KNN (K-Nearest Neighbors), and Decision Trees to create a machine learning model that could predict the genre of a song given various metrics of that song such as the length of the song, the number of instrument, and tempo changes.
